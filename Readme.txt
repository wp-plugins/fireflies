== Description ==

This is another revision of the WordPress plugin "Hello Dolly" that displays Owl City's song "Fireflies" on the top right corner of the WordPress Dashboard. 

== Installation ==

There are two ways to install the Fireflies plugin:

1) Install it from the WordPress Plugin program
2) Upload fireflies.zip into WordPress from http://www.winworldblog.com/
